# Showdown Replay Studio (v1.1)

An _actual_ Pokémon Showdown Replay Downloader.

📥Download any PS replay via its link into **mp4/webm** format.

Utilise as a local web application with various customization options and the ability to batch download replays simultaneously.

Additionally you can download locally saved HTML file replays into mp4 recordings as well.

## Setup

Join the [discord server](https://discord.gg/dvcbNJu5zS) for any setup queries!

### Windows users:

1. No Setup required!
2. Head to Releases and download the latest release for your Operating System. Unzip the folder.
3. Open the `Showdown Replay Studio.exe`

### For macOS & Linux users:
Clone this repository (i.e. download all files).

You can click on Code -> Download zip and then unzip the downloaded project.

1. Install Node.js (https://nodejs.org/en/download)
2. Open terminal at the project directory's path and enter `npm install` (example:- `/home/intenzi/Showdown Replay Studio> npm install`)
3. Enter `npm start` to start the application.

> [!NOTE]
> Step 1, Step 2 to be performed only once. Next time onwards its solely step 3 to start the app.
